jQuery.validate使用方法

自带验证规则：

  1. number 验证全数字

  2. email 验证电子邮箱

  3. empty 验证非空

  4. phone 验证手机号码

使用方法：

效果：

代码：

validate({ 调用验证
id:"na", 验证器名称,方便以后调用
tipPlacement:function(element,tip){ 提示信息放置函数
element.closest('.common').find('.tip').html(tip);将提示信息放在哪里，element为input节点，tip为提示内容
},
rules:{
'a':{ name为a的需要验证的节点
wrong:"请输入中文", wrong,right,empty,focus这几个字段为提示内容
right:"输入正确",
empty:'输入不能为空',
focus:"请您输入该字段",
required:true, 是否为必填
limit:/^[\u4e00-\u9fa5]+$/ 验证正则
}
}
});
